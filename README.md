### Installation

#### Clone repo

``` bash
# clone the repo
$ https://github.com/rugube/BulkSMS-Platform.git

# go into app's directory

# install app's dependencies
$ npm install
```

#### Usage

``` bash
# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build

# run linter
npm run lint

# run unit tests
npm run test:unit

# run e2e tests
npm run test:e2e

```
